# Cyprus Villa

Премиальный одностраничный сайт-экскурсия для аренды виллы на Кипре.

## Структура

- `index.html` — разметка кинематографического маршрута.
- `src/styles.css` — визуальная система, адаптив, анимации и performance-friendly состояния.
- `src/main.js` — управление сценами, плавным скроллом, видео и формой.
- `assets/video` — локальные оптимизированные версии исходного видео.
- `assets/gallery` и `assets/posters` — кадры, извлеченные из локального видео.

## Локальный запуск

```bash
node server.mjs
```

Затем открыть `http://localhost:4173`.
